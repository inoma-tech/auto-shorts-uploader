AutoShortsUploader

AutoShortsUploader is a developer tool that turns a single CSV line into a fully-edited TikTok / YouTube Shorts video and posts it via TikTok’s official Content Posting API – all in under 60 seconds.

Key features

  • CSV → Speech → Video → Direct Post – one command end-to-end.
  • Uses ElevenLabs TTS, Hedra face animation, and ffmpeg templates.
  • Supports batch queues & GitHub Actions for hands-free posting.
